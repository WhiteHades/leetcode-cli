# LeetCode CLI Documentation

Welcome to the **LeetCode CLI** documentation.

## Overview

A modern, feature-rich command-line interface for LeetCode, built with TypeScript.

## Features

- 🔐 Cookie-based authentication
- 🌐 Site-aware API support (`leetcode.com` and `leetcode.cn`)
- 📝 Auto-generation of solution files
- 🧪 Local testing against sample cases
- 📤 Direct submission to LeetCode
- 🖥️ Interactive Terminal UI (TUI) mode
- 🐳 Docker support
